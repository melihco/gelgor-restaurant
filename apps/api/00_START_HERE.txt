GETTING STARTED (5 MINUTES)
================================================================================

1. Create PostgreSQL Database:
   createdb nexus_api

2. Start Redis:
   redis-server
   (or: docker run -d -p 6379:6379 redis)

3. Build Solution:
   cd /sessions/busy-dazzling-gates/mnt/SmartAgency/apps/api
   dotnet build

4. Run API:
   cd src/Nexus.Api
   dotnet run

5. Open in Browser:
   http://localhost:5000

See QUICKSTART.md for detailed instructions.
